## Introduction 
Spring Boot bare-bone OAuth Server for demo purpose. Not suitable for production environment

You need 
* Java JDK 8 or higher
* Maven 3.1.1 or higher

## Building
To build run following 

```bash
$ mvn clean install
```

### Run Local
```bash
$ mvn spring-boot:run
```
